# Lab 03 – Descriptive Statistical Analysis

## Aim
The aim of this lab is to apply statistical techniques to summarize, explore, and interpret datasets before applying machine learning models. Using Python libraries such as **NumPy, Pandas, SciPy, and Matplotlib**, the lab focuses on computing descriptive statistics like mean, median, mode, variance, standard deviation, quartiles, correlation, and covariance to understand the distribution and relationships within data.

## Dataset

**Dataset Name:** Students Performance Dataset

**Dataset Link:**  
https://github.com/rashida048/Datasets/blob/master/StudentsPerformance.csv

This dataset contains information related to students’ academic performance along with demographic attributes.

### Features in the Dataset
- gender
- race/ethnicity
- parental level of education
- lunch
- test preparation course
- math score
- reading score
- writing score

The dataset is widely used for **educational data analysis** and helps in understanding relationships between demographic factors and student performance.

## Libraries Used
- Python
- Pandas
- NumPy
- SciPy
- Matplotlib
- Seaborn

## Experiments

1. Compute measures of **central tendency** (mean, median, mode) and **dispersion** (min, max, variance, standard deviation).
2. Calculate **quartiles (Q1, Q2, Q3)** and interpret their significance.
3. Compute **correlation and covariance** between numerical variables.
4. Visualize data distribution using **histograms and boxplots**.

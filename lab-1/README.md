# Lab 01 – Introduction to Python for Data Analysis

This lab introduces the basics of Python used for data analysis. The experiments focus on using libraries such as **NumPy** and **Pandas** to create and manipulate arrays, generate matrices, and analyze datasets. Students also learn how to read CSV files and explore data using functions like `head()`, `tail()`, `info()`, and `describe()`.

## Experiments Included
1. Creating a NumPy array from a Python list and displaying its shape and data type.
2. Creating matrices of zeros and ones and displaying their shapes.
3. Creating an array of numbers and reshaping it into a matrix.
4. Generating random integers and finding minimum, maximum, mean, and sum.
5. Creating a Pandas DataFrame using a dictionary and displaying column data types.
6. Reading a CSV file and displaying the first and last records.
7. Loading a dataset and displaying dataset information such as rows, columns, and missing values.
8. Reading a CSV file and displaying statistical summary using `describe()`.

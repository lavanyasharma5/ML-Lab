# Lab 02 – Data Preprocessing and Visualization

## Aim
The aim of this lab is to perform **data preprocessing and visualization** to prepare datasets for machine learning tasks. It involves identifying missing values, cleaning datasets, transforming data types, and visualizing data to understand patterns and relationships.

## Dataset
The dataset used in this lab is stored in this repository.

**File:** `data.csv`

The dataset is used to practice:
- Handling missing values
- Cleaning inconsistent data
- Renaming columns
- Converting data types
- Data visualization using plots

## Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Experiments
1. Identify missing values in a dataset.
2. Remove rows containing missing values.
3. Replace missing values using mean, median, or constants.
4. Display and convert column data types.
5. Rename column names for better readability.
6. Identify and correct inconsistent values.
7. Create line plots and bar plots.
8. Create scatter plots between numerical features.
9. Plot histograms to analyze data distribution.
10. Create boxplots to identify outliers.
